# Session Template

`LEARNING_MEMORY.md` を更新するときは、この型を使います。

```md
### YYYY-MM-DD

#### 今日取り組んだこと

- 

#### 自分でできたこと

- 

#### つまずいたこと

- 

#### まだ曖昧なこと

- 

#### 次回のCodexのサポート方針

- 
```

## 記録例

```md
### 2026-06-23

#### 今日取り組んだこと

- `works/work1/q3.php` の偶数・奇数判定を確認した。

#### 自分でできたこと

- `if / else` の形で条件分岐を書こうとしていた。

#### つまずいたこと

- 割り算の結果と、割った余りの違いが曖昧だった。

#### まだ曖昧なこと

- 偶数判定でどの演算子を見るべきか。

#### 次回のCodexのサポート方針

- 答えを出さず、まず「余り」に注目するヒントを出す。
```
