# Interaction Log

このファイルは、システム構築以外の学習相談を記録するログです。

## 記録ルール

- 対象: ヒント依頼、エラー相談、考え方の相談、答え合わせ、評価依頼。
- 除外: `AGENTS.md`, `LEARNING_MEMORY.md`, `docs/learning/`, `learning-memory/` などの仕組み作り。
- 完成コードや最終解答そのものは記録しない。
- 次回の支援に使えるよう、短く具体的に書く。

## 記録テンプレート

```md
### YYYY-MM-DD

- 種類:
- 対象:
- 質問の要約:
- 支援内容:
- つまずき:
- 理解できていること:
- 次回の支援方針:
```
