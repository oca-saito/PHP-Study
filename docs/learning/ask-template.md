# Codexへの聞き方テンプレート

毎回長く説明しなくても、次のように短く聞けば大丈夫です。

## ヒントだけほしい

```text
works/work1/q3.php のヒントだけください。答えは出さないでください。
```

## 考え方を確認したい

```text
この問題の考え方だけ教えてください。完成コードは不要です。
```

## 自分のコードのどこが違うか知りたい

```text
works/work1/q3.php の今のコードを見て、どこで考え方がずれているか教えてください。答えは出さないでください。
```

## エラー文を読みたい

```text
このエラー文の読み方を教えてください。修正コードではなく、原因の探し方を知りたいです。
```

## 答え合わせしたい

```text
自分で書いたので、考え方が合っているか答え合わせしてください。必要な指摘だけお願いします。
```

## 実行確認したい

```text
このファイルの実行方法と、見るべき出力を教えてください。
```
